# Navbar

A clean and responsive navigation bar component built with Tailwind CSS and JavaScript.

## Features

- Responsive layout
- Mobile navigation toggle
- Semantic HTML structure
- Accessible navigation button (`aria-label`)
- Keyboard focus styles
- Hover effects
- Mobile-first design
- Built with utility-first Tailwind CSS

## Technologies

- HTML5
- Tailwind CSS
- JavaScript
