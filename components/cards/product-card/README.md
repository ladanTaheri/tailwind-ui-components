 ## Product Card

A responsive and reusable product card component built with HTML and Tailwind CSS. Designed for e-commerce interfaces with a clean layout, fixed image container, and accessible action button.

## Features

- Responsive design
- Semantic HTML structure
- Fixed image container for consistent card layout
- Product image displayed with `object-contain`
- Keyboard focus state
- Hover effects
- Accessible icon using `aria-hidden`
- Built with Tailwind CSS

## Technologies

- HTML5
- Tailwind CSS
- Font Awesome
