# Profile Card

A responsive and modern profile card component built with HTML and Tailwind CSS. Designed with a clean layout, circular avatar, user information, and responsive info cards.

## Features

- Responsive design
- Semantic HTML structure
- Circular profile avatar
- User information section
- Responsive info cards
- Dark mode support
- Accessible icons using `aria-hidden`
- Built with Tailwind CSS

## Technologies

- HTML5
- Tailwind CSS
- Font Awesome
