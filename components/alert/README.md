# Info Alert

A clean and responsive information alert component built with Tailwind CSS.

## Features

- Responsive layout
- Accessible icon (`aria-hidden`)
- Keyboard focus styles
- Hover effects
- Rounded corners
- Font Awesome icons
- Clean semantic HTML
- Built with utility-first Tailwind CSS

## Technologies

- HTML5
- Tailwind CSS
- Font Awesome
