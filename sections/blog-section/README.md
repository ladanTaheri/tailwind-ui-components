# Blog Section

A responsive blog section built with HTML and Tailwind CSS. It features a clean layout with article cards, category badges, featured images, and call-to-action links for showcasing blog content.

## Features

- Responsive layout
- Semantic HTML structure
- Reusable article card design
- Category badges
- Featured images with consistent sizing
- Hover animations and shadow effects
- Accessible links with focus states
- Built with Tailwind CSS

## Technologies

- HTML5
- Tailwind CSS
- Font Awesome
